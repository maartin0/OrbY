# OrbY
**Orb**ital Displa**y**<br>
A solar system viewer written in [three.js](https://github.com/mrdoob/three.js/)

## Planets
Currently, the following planets are simulated:
* The Sun
* Mercury
* Venus
* Earth
* Mars
* Jupiter
* Saturn
* Uranus
* Neptune
* Pluto
